# apps/deck — AIOS jury deck

A Slidev project for the hackathon jury presentation.

## Why Slidev

- Markdown-sourced → versioned with the code it describes
- Embeds live components (iframe into `http://localhost:5173`, Mermaid, charts)
- PDF export for handout, presenter mode for live delivery
- Stays local — no data leaves the box (VISION.md §3)

## Setup

1. Install the cc-slidev Claude Code plugin:

   ```
   /plugin marketplace add rhuss/cc-rhuss-marketplace
   /plugin install slidev@cc-rhuss-marketplace
   /reload-plugins
   ```

2. From this directory, scaffold the Slidev project:

   ```
   cd apps/deck
   /slidev:init
   ```

3. Import the outline + Stripe theme:

   - Feed `OUTLINE.md` to `/slidev:generate` to seed `slides.md`.
   - Import `theme/stripe/styles.css` from the top of `slides.md`:

     ```yaml
     ---
     css: ./theme/stripe/styles.css
     ---
     ```

4. Preview and export:

   ```
   /slidev:preview           # http://localhost:3030
   /slidev:export pdf        # apps/deck/slides-export.pdf
   ```

## Files

| Path | Purpose |
|---|---|
| `OUTLINE.md` | Slide-by-slide plan mapped to VISION.md §9 |
